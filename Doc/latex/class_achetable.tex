\doxysubsection{Description détaillée}
La classe \mbox{\hyperlink{class_achetable}{Achetable}} permet d\textquotesingle{}abstraire une case achetable quelconque du plateau de jeu. La classe \mbox{\hyperlink{class_achetable}{Achetable}} est fille de la classe \mbox{\hyperlink{class_case}{Case}}. Cette classe n\textquotesingle{}a pas pour but d\textquotesingle{}être utilisée telle quelle, il s\textquotesingle{}agit en fait d\textquotesingle{}une classe mère de laquelle découleront les véritables cases qui constituerons notre plateau.

Les classes découlant de \mbox{\hyperlink{class_case}{Case}} sont \mbox{\hyperlink{class_gare}{Gare}}, \mbox{\hyperlink{class_constructible}{Constructible}} et \mbox{\hyperlink{class_service}{Service}}. Nous notons l\textquotesingle{}implémentation de la méthode is\+Achetable() qui permettra de différencier ces les objets achetables des non achetables. Nous notons la méthode purement virtuelle is\+Achetable() qui permettra de différencier ces deux classes dans notre implémentation.
