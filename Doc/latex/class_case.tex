\doxysubsection{Description détaillée}
La classe \mbox{\hyperlink{class_case}{Case}} permet d\textquotesingle{}abstraire une case quelconque du plateau de jeu. Cette classe n\textquotesingle{}a pas pour but d\textquotesingle{}être utilisée telle quelle, il s\textquotesingle{}agit en fait d\textquotesingle{}une classe mère de laquelle découleront les véritables cases qui constituerons notre plateau.

Les deux grandes classes découlant de \mbox{\hyperlink{class_case}{Case}} sont \mbox{\hyperlink{class_achetable}{Achetable}} et \mbox{\hyperlink{class_non__achetable}{Non\+\_\+achetable}}. Nous notons la méthode purement virtuelle is\+Achetable() qui permettra de différencier ces deux classes dans notre implémentation.
