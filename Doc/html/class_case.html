Description détaillée

La classe Case permet d'abstraire une case quelconque du plateau de jeu. Cette classe n'a pas pour but d'être utilisée telle quelle, il s'agit en fait d'une classe mère de laquelle découleront les véritables cases qui constituerons notre plateau.

Les deux grandes classes découlant de Case sont Achetable et Non_achetable. Nous notons la méthode purement virtuelle isAchetable() qui permettra de différencier ces deux classes dans notre implémentation.
